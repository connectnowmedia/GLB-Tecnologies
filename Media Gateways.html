Service Details

Media gateways are devices that help connect different communication networks, such as the traditional Public Switched Telephone Network (PSTN) and modern IP-based networks. These gateways act as intermediaries between different communication protocols and allow seamless communication between networks that may not natively support each other. Media gateways can convert signals between different types of networks and translate communication protocols to ensure smooth and efficient communication. They are essential components in today's communication infrastructure and play a vital role in enabling communication between various networks and devices.

The IRIS IVDX Cloud – 1U Module is a powerful and versatile hardware solution that can function as a standalone IP-PBX with SIP switching or as a media gateway when registered to a central Soft-switch. It is built with IMS architecture in mind and offers multiple protocols for seamless connectivity and ease of operation. The IRIS IVDX Cloud – 1U Module is designed to provide connectivity to all legacy and next generation telecom networks such as POTS, PSTN, ISDN, E1/PRI, GSM and IP in a single box. This makes it an ideal solution for networks requiring interworking with multiple legacy and new generation protocols for enterprise communication. It offers an easy migration path from traditional to IP communication, making it a valuable investment for organisations looking to upgrade their communication infrastructure. The IRIS IVDX Cloud – 1U Module is reliable, innovative, and scalable. It integrates voice, video, and business-enhancing features such as Unified Communications with integrated Voice Mail, Contact Center, and more. This solution is particularly well-suited for high-security networks, such as defence communication, where security and reliability are of paramount importance. The IRIS IVDX Cloud – 1U Module provides an unparalleled level of resilience and can be cascaded for even greater connectivity and redundancy. Choose the IRIS IVDX Cloud – 1U Module for a seamless transition to IP-based communication that meets all your enterprise needs.

A Gateway is a device that enables analog telephony devices to connect to an IP network. Our Gateway product line offers multiple options, including 32/48/64/96 ports, with built-in SIP licences and trunks for seamless integration with other gateways or IP PBXs. Features include support for up to 96 analog FXS ports, a 4-port Ethernet switch with routing capabilities, and compatibility with multiple protocols such as SIP V2.0, IMS/3GPP, SDP, RTP/RTCP, MGCP, and H.248. The device also supports web-based administration for easy remote maintenance. Applications for our Gateway products include low-cost media gateway for TDM to IP conversion, shared use by multiple telecom operators to provide analog telephony services at a location, and support for switching across multiple gateways for basic telephony services without a soft-switch using routing features/MGCP. Our Gateway product line is fully compliant with SIP standards and interoperable with various IP Telephony equipment, including IPPBXs, Gateways, and Phone systems. It is also designed to meet the requirements of network applications in different network environments, supporting voice, fax, and other services using existing cabling infrastructure.

IRIS IVDX Cloud – 3U Module is a versatile hardware solution that can function as a standalone IP-PBX with SIP switching or as a media gateway when registered to a central Soft-switch. This 3U module offers connectivity to all legacy and next generation telecom networks, making it an ideal solution for enterprises seeking easy migration from legacy systems to current and new-age IP based infrastructure.The IRIS IVDX Cloud – 3U Module supports multiple protocols, providing ease of operation, and is designed with IMS architecture in mind. It can cascade for seamless connectivity with adjacent similar switches, providing immense resilience. This hardware integrates voice, video and is loaded with business-enhancing features such as Unified Communications with integrated Voice Mail, Contact Center, and more. The IRIS IVDX Cloud – 3U Module is ideal for high availability communication networks for Utility services, Metro projects, Railways, Large PSU’s, networks requiring interworking with multiple legacy and new generation protocols for enterprise communication, and high-security networks like Defense communication. Its high-density media gateway provides redundancy, making it an ideal solution for enterprises seeking a reliable and scalable solution for their communication needs.
